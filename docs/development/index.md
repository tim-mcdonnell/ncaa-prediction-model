## Getting Started with Development

New developers should start by:

1. Reading the [AI Coding Agent Guide](./ai-coding-agent-guide.md) for a high-level overview
2. Reviewing the [Pipeline Architecture](./pipeline-architecture.md) to understand the system design
3. Familiarizing themselves with the [Project Structure](./project-structure.md)
4. Understanding the [Test Strategy](./test-strategy.md) for development workflow

For task-specific work, refer to the individual milestone documentation and task descriptions in the respective directories.
